# 🚨 Alerta Rural - Sistema de Emergencias Comunitarias

Progressive Web App para notificación y gestión de incidentes en zonas rurales de Chile.

## 🛠️ Tecnologías
- React 18 + Vite
- Firebase (Authentication, Firestore, Hosting)
- Web Speech API + Geolocation API
- CSS Nativo (Mobile-First)
